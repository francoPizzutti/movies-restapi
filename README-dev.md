# Useful commands

## Doctrine:
Migrate database
```sh 
php bin/console doctrine:migrations:migrate  
```
---

## Maker bundle:

### Create migration:
```sh 
php bin/console make:migration
```

### Create/edit entity w/relationships:
```sh 
php bin/console make:entity $entityname
```
--- 
## Start symfony server
```sh
symfony server:start
```
---
